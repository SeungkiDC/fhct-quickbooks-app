FHCT Bookkeeping Automation — Application Information

Operated by: Friends Healthcare Team, Inc.

Contact: david@friendshealthcareteam.com

About this application

FHCT Bookkeeping Automation is a private, internal tool used solely by Friends Healthcare Team, Inc. to read its own QuickBooks Online financial data through Intuit’s authorized API. It assists the company’s accounting team with bank reconciliation, categorization of unposted transactions, and accrual preparation. The application uses read-only access and is not offered to or used by any third party.

Connecting

The application is connected to QuickBooks Online by an authorized Friends Healthcare Team administrator through Intuit’s standard OAuth authorization. Because this is a private, internal integration, there is no public sign-up or login.

Reconnecting

If the connection expires or is revoked, an authorized administrator re-authorizes the application through Intuit’s standard OAuth flow.

Disconnecting

Access can be disconnected at any time by the administrator from within the Intuit account (Settings → Apps / Connected apps), or by revoking the authorization. Upon disconnection, the application ceases to access QuickBooks Online data.
